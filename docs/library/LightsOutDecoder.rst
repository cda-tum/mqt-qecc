LightsOut Decoder
=================

QECC provides means to decode quantum codes based on an analogy to the well-known LightsOut puzzle.

.. currentmodule:: mqt.qecc.cc_decoder.decoder

The main class for the decoder is the :class:`LightsOut` class.
